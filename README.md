\# 🏋️‍♂️ OmniFit — The Omniscient AI Personal Trainer



OmniFit is a state-of-the-art Android fitness companion built using \*\*Kotlin\*\* and \*\*Jetpack Compose\*\*. It blends offline skeletal feedback with advanced generative AI to deliver interactive, highly visual, and safe coaching.



\## ✨ Key Features

\*   \*\*🤖 AI Form Guard Engine:\*\* Real-time camera feed overlay utilizing custom 2D vector calculation to track joint angles (Squat depth, Push-Up sag, Bicep Curl range) accurately on-device.

\*   \*\*🌟 Smart Coaching with Gemini:\*\* Integrated conversational fitness mentor that drafts custom exercise regimens and answers nutritional queries instantly.

\*   \*\*📊 Dynamic 3D Muscle Heatmap:\*\* Interactive visual mapping tracking concentric muscle recovery and active muscle strain.

\*   \*\*📅 Workout \& Nutrition Planners:\*\* Automated schedule generators keeping your fitness calendar and caloric targets aligned.



\## 🛠️ Tech Stack

\*   \*\*UI Framework:\*\* Jetpack Compose (Material Design 3)

\*   \*\*Camera Integration:\*\* Android Jetpack CameraX (ProcessCameraProvider \& SurfaceProvider)

\*   \*\*AI Integration:\*\* Google Gemini API

\*   \*\*Architecture:\*\* MVVM Design Pattern with clean state flows

\*   \*\*Database:\*\* SQLite Room persistence
